# Wendyが認識する音声用辞書ファイル群

 - wendystart.conf  
  wendy辞書を使ってjuliusuを起動させるためのconfigファイル
 - wendy.*
  julisuが必要とする辞書ファイル

## 認識できる音声  

 - __おきて__ (delight。待機状態⇒音声認識待ち状態へ。importantなワード)
 - __さようなら__ (sleepy。音声認識待ち状態⇒待機状態へ。importantなワード)
 - __おやすみ__ (待機状態で有効。ラズパイを終わらせる。importantなワード)
 - ウエンディ
 - まさお (delight）
 - きつね (delight）
 - とまれ (stop）
 - ひだり (left)
 - みぎ (right)
 - すすめ (go)
 - おはよう (sleepy)
 - こんにちは (sleepy)
 - こんばんは (sleepy)
 - さようなら (sleepy)
